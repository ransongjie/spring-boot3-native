# Native Image
云原生技术
.java 到 可执行文件(.exe), 代码到可执行文件, 运行时不再需要jdk

降低运行成本
方便容器部署
加快启动速度
初始最佳性能

spring boot 3.0+
graalvm-ce-java17-22.3.0, 自带jdk-17
